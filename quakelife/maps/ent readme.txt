QuakeLife Half-Life 1 Map Ents

-=[ Usage ]=-
1) Place .ent and .cfg files in the same folder as half-life maps
2) Execute supplied batch file
3) Start with FTE/Darkplaces*
* Darkplaces support may vary on version of mod supplied
